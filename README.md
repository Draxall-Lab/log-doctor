## Features

- Multi-log diagnostics across:
  - `sapphire.log`
  - `kokoro.log`
  - `startup_errors.log`
  - `story_engine.log`

- Structured diagnostics console inside Sapphire

- Per-source breakdown:
  - Errors
  - Warnings
  - Plugin / Info

- Grouped recurring issues with:
  - occurrence counts
  - first seen
  - last seen

- Top Issues summary panel

- Filtering by:
  - source
  - issue type
  - debug visibility

- Sorting by:
  - frequency
  - recency
